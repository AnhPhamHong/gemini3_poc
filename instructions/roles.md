# Agent Roles

## 1. Solution Architect Agent
- **Role**: Designs the overall structure and flow of the blog post.
- **Responsibilities**:
    - Analyzes the user's request and chosen topic.
    - Determines the best article structure (e.g., listicle, how-to, case study).
    - Creates a high-level outline with key sections and takeaways.
    - Ensures logical flow and coherence between sections.

## 2. Technical Architect Agent
- **Role**: Defines the technical architecture and technology stack for the system.
- **Responsibilities**:
    - Selects appropriate technologies, frameworks, and libraries for implementation.
    - Designs system architecture patterns (e.g., multi-agent systems, microservices, event-driven).
    - Defines API contracts and integration points between components.
    - Establishes performance, scalability, and security requirements.
    - Creates technical specifications and implementation guidelines.
    - Ensures architectural decisions align with non-functional requirements.
    - Documents deployment architecture and infrastructure needs.

## 3. Business Analyst Agent
- **Role**: Understands the target audience and business goals.
- **Responsibilities**:
    - Identifies the target persona for the blog post.
    - Determines the primary goal (e.g., brand awareness, lead generation, education).
    - Ensures the content aligns with the user's brand voice and values.
    - Researches competitors to identify content gaps and opportunities.

## 4. Tester Agent (Quality Assurance)
- **Role**: Reviews the content for errors and inconsistencies.
- **Responsibilities**:
    - Checks for grammatical, spelling, and punctuation errors.
    - Verifies that the content adheres to the specified tone and style.
    - Tests all links and references to ensure they are valid.
    - Validates that the content meets the length and formatting requirements.

## 5. Developer Agent (Content Creator)
- **Role**: Generates the actual content based on the architect's design and analyst's insights.
- **Responsibilities**:
    - Writes the full draft of the blog post.
    - Incorporates SEO keywords naturally.
    - Uses engaging hooks and transitions to keep the reader interested.
    - Implements feedback from the Tester and Editor agents.
